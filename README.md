# Python Əsasları - Jupyter Notebook Qeydləri

Bu repository Python proqramlaşdırma dilinin əsas mövzularına dair şəxsi qeydlərimi və kod nümunələrini ehtiva edir. Hər bir mövzu ayrıca `.ipynb` faylında izah olunub və praktiki nümunələrlə dəstəklənib.

## Fayl Strukturu

- **Calculator.ipynb**  
  Sadə kalkulyator nümunəsi (toplama, çıxma, vurma, bölmə funksiyaları).

- **Factorial.ipynb**  
  Faktorial hesablayan funksiyalar (for, while və rekursiv üsullar).

- **Fibonacci.ipynb**  
  Fibonacci ardıcıllığını hesablayan nümunələr.

- **For_While.ipynb**  
  `for` və `while` döngülərinə aid nümunələr və izahlar.

- **Functions.ipynb**  
  Funksiyalar, parametr növləri, `return` istifadəsi.

- **List_Comprehension.ipynb**  
  List comprehension sintaksisi və nümunələri.

- **Metods.ipynb**  
  Python obyekt metodları (string metodları, list metodları və s.).

- **Modules.ipynb**  
  Python modullarının (`math`, `random`, `datetime` və s.) istifadəsi.

- **Tuple.ipynb**  
  Tuple məlumat strukturu, xüsusiyyətləri və istifadəsi.
